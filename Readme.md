# Agentic AI Workspace

This workspace contains the **Agentic AI Bill Splitter** project.

## Projects

### [Bill Splitter](./BillSplitter)
A multi-agent system for automating monthly wireless bill splitting, expense tracking, and notifications.

**Key Features:**
*   Gemini 2.0 Flash for PDF parsing
*   LangGraph Orchestration
*   Splitwise MCP Integration
*   WhatsApp Notifications

[View Bill Splitter Documentation](./BillSplitter/README.md)
